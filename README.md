# Visualizing Agricultural Meta-Analysis: N2O Emissions vs. Crop Yield

## Overview
This repository contains the Python scripts and resulting data visualizations for a comprehensive meta-analysis examining the impact of various agricultural practices on N2O emissions and crop yield. 

## Technical Stack
* **Language:** Python
* **Libraries:** Pandas (data structuring), Matplotlib & NumPy (advanced data visualization)
* **Output:** High-resolution (600 DPI) publication-ready forest plots.

## Visualizations Included
1. **Overall Impact:** Grand mean comparisons of Risk Ratios (RR).
2. **Fertilizers & Nitrogen:** Stratified analysis by application method, rate, and source.
3. **Agricultural Practices:** Breakdown by irrigation, tillage, land use, and climate zones.
4. **Soil Properties:** Analysis based on soil pH, texture, and SOC.

## Methodology
The visualizations utilize custom Matplotlib styling to generate side-by-side forest plots. The code structures raw meta-analytical data (sample sizes, confidence intervals, means) into a highly readable format, integrating grand means and sub-category dividing lines for academic publication standards.
